## üéØ **Features**

Zcrypt Academy is built around four core modules:

---

### **1. ZK Book (Course)**

A deep dive into Zero-Knowledge Proofs, based on the ZKProof Community Reference.

Includes:

* Security foundations: *Completeness, Soundness, Zero-Knowledge*
* Paradigms: *Interactive Systems, Cryptographic Compilers, Arithmetization (R1CS, QAP)*
* Implementation: *Frontends, Backends, APIs, Benchmarks*
* Applications:

  * Identity & Authentication
  * UTXO / Balance Model Transfers
  * Privacy-Preserving Compliance

---

### **2. Rust Mastery (Course)**

A complete Rust learning path, essential for ZK system development.

Topics include:

* rustup, cargo, toolchains
* Variables, mutability, primitives
* Ownership & borrowing
* Control flow
* Structs, enums, traits
* Concurrency & async

---

### **3. Ecosystem Pulse (Blog)**

A curated feed of high-signal content from the privacy and crypto world.

Sources include:

Topics are categorized (Scaling, Architecture, Research) for easy browsing.
